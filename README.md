# 🌿 Living Standards Initiative (LSI) India

> **Mapping What Official Data Forgets.**

India has 640,000+ villages — but most don't exist in any meaningful dataset. LSI is a student-led initiative documenting real living conditions — electricity, water, healthcare, education — village by village. Because if a place isn't visible, it can't be prioritized.

---

## 🎯 Mission

Bridge India's rural data gap by conducting ground-level surveys and building an open, accessible dataset of living standards across villages and underserved communities. We believe **visibility is the first step to change**.
